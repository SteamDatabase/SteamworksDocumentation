BAFTA Review Package

Over the past several years, Steam has created a "BAFTA Review Package" to streamline the process for BAFTA members to access and review games that are entering for BAFTA consideration.

If you have entered your game for BAFTA consideration and would like it included in the BAFTA Review package, please submit a ticket under Steam Seasonal Sales or Sale Events.

Please include the following:
  • "BAFTA Review Package"
  • The App ID & game name to be included
  • Any notes to confirm what content for the game that should or should not be included. For example:
    • Include all OS versions
    • Include Soundtrack and/or additional DLC (please include App IDs)
    • Do not include beta depot ID
Note: This package is only used to provide the official BAFTA judges with access to approved nominees. Valve does not officially manage BAFTA voting or nomination. See the BAFTA website for more information or to submit your game to the BAFTAs for consideration as a nominee.
