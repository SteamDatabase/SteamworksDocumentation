Working with a publisher

Overview

If you are a small team or individual game developer looking to work with a publisher for funding, marketing, testing, or any other services, there are some useful things to consider and plan for. This documentation is a collection of resources and FAQs on the ways that working with a publisher might impact how your game is set up and managed within Steamworks.

Game Access Within Steamworks

When multiple entities (such as a developer team and a publishing company) need access to the same application within Steamworks, some special configuration will be necessary to make sure the right permissions are defined. For a complete break-down on the ways access can be set up, please see Managing Applications.
