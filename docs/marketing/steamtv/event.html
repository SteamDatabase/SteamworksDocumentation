SteamTV Setup

Note: This is a BETA feature. It may change at any time or it may be discontinued.

Event Setup

Setting up SteamTV is manual process. The Steam team requires the following information to setup your event on Steam.tv:

  1. 1920x1080 splash screen that includes the logo for the event, to be shown when the stream is inactive. An example, is from the CSGO London Major.
  2. 30x30 chat tab icon (typically includes the event logo). An example, is from the CSGO London Major.
  3. A broad schedule (days for main event, group stage etcâ€¦)
  4. What time of day will the streaming start? Is that the same each day?
  5. The number of parallel streams and the SteamID that will be used to broadcast those streams (we need to white list them)
  6. If there are parallel streams, what are their purpose, so that we can describe them in drop-downs
    1. Parallel matches?
    2. Are they different language streams?
    3. Beginner streams?

Account Setup

You need to use a Steam account to stream to SteamTV via the existing Steam Broadcast infrastructure. Here are the requirements for that Steam account.

  1. The broadcasting account needs to be non-limited; must have $5 USD spend on Steam
  2. The broadcasting account MUST have in its Steam library the game you plan on stream.
  3. The broadcasting account needs to join the Store Broadcast Beta community group to gain access to creating an RTMP token.
  4. While logged into the broadcasting account, you can determine the best RTMP server and finish the remainder of the setup here.
