Steam's Summer Showcase of Summer Showcases

Steam's hub for the summer showcase events (e.g. Summer Game Fest, PC Gaming Show, Xbox Showcase) happening around the weekend of Summer Game Fest. In this hub, players will find links and highlighted games from each of the participating showcase landing pages.

Likely around June 10-17; Final dates to be announced once Summer Game Fest is scheduled.

Updated for 2026

This hub on Steam will be primarily a list of each of the participating events, along with a carousel of top wishlisted games from that event, and a clickable banner leading players to that event page.

Each individual event page is built and managed by the organizers of that event, and can host a livestream and lists of the announced games.

Steam's hub page itself will not be hosting the livestreams.

Important Dates

These are the important milestones we need to hit along the way
  • March 15, 2026 - Do you intend to participate with an event? We need to know. You'll need to fill out the Submission Form at the bottom of our information and rules page. Get started here: https://partner.steamgames.com/doc/store/promo/organizers
  • May 15, 2026 - Does your livestream work? Make sure you've tested the livestream on your Steam event page
  • June 1, 2025 - Has your event page been submitted for review? Make sure you give us enough time to review your event page and approve it for publishing
  • Early June - Kick Off! - The show kicks off with the first livestream. Valve will send out a press announcement about our Summer Showcase hub page. Hub landing page will become visible, showing the schedule of livestreams and grid of participating events.

How to Participate

This Summer Showcase is built around the idea of multiple events that are each showcasing a set of new game announcements or updates, unveiled via livestream. To qualify as part of the Steam Summer Showcase, each participating event must be running a livestream at some point with major new announcements.

Event Organizer Requirements

To participate, each event must provide Steam with the following

  • Let us know that you want to participate by carefully reading the rules and filling out the submission form linked at the bottom of the page here: https://partner.steamgames.com/doc/store/promo/organizers.
  • The Steam promo team will help enable the landing page editor tools, and will send you an asset request with templates and requirements for the assets we need to be able to host and promote your event fully.
  • Once the landing page editor tools have been enabled, you'll need to configure your landing page on Steam, adding your event info, background image, logo, and sections for the announced games.
  • You need to complete the asset request form by the noted deadline, sending us a layered copy of artwork you use on Steam for your event page, including any non-standard fonts.
  • You also need to submit your landing page itself for review by the deadline noted above so that our promo team can make sure everything looks configured correctly.

Create a landing page on Steam

Each participating event will have a landing page where you'll be livestreaming and showcasing the games that are being announced. To get started creating this page, please visit https://partner.steamgames.com/doc/store/promo/organizers, read the rules, and click the link at the bottom to get started building your page. When you fill out the form, make sure to include the fact that this is part of "Summer Showcases".

This landing page should have clear branding and description of who is organizing the event, what the event is, and specific dates and times that players would be most interested in (such as any unveiling livestream, or new game playthroughs, etc.)

Layered Artwork and Font Files

We'll be translating many of the promo materials into the standard set of 26 supported languages on Steam, so we need your layered files to make sure we can include your event in this process. Here is what we need:
  • A layered Photoshop (PSD) file that includes your event logo (smart object preferred) separated from the art. Any text should be editable in a separate “text” folder within the PSD document, so that it may be easily localized by our team.
  • Please send us these layered files for any of the banners and images you are including on your sale page and livestream event. This includes Cover Image, Sale Page Background Image, Store Page Banners, and any images you include on your sale page that contain text.
  • Please also include any non-standard fonts used

Frequently Asked Questions

Q. What if I'm not ready to set my landing page live by the announcement date?
A. Your event will not appear until you have set your landing page visible. You can set your landing page to be visible, even if the start date of the event is in the future.
