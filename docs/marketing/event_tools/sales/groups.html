Sale Page Tools - Background Groupings

Background groupings allows you to configure background visuals that span one or more section of your sale page. These backgrounds span the full width of the page, and let you apply various formatting of images and/or gradient colors.

Before you start

In order to make use of this feature, you will need to:
  • Review the rules and guidelines for the type of event you are running.
  • At the bottom of each rules page, find the link under "Getting started" and submit a support ticket with information about your event.
  • Once enabled, you'll need to add your games, configure sections and finish most of the merchandising work to organize the games in your sale.
  • Flip over to the 'preview' of your sale page, and you'll see the controls to enable background groupings.

Using the Background Groupings

Enabling Groupings

To start using groupings, you'll need to be in preview mode of your sale page. Then select "Enable Groupings" in the upper-right corner. You will see some additional UI appear with further options.

You can de-select the "Edit mode" checkbox to hide the editor UI if you need cleaner preview of your work.

Adding New Groupings

Groupings are possible starting with the first content section of your sale page. This assumes that you are already using the sale page background and sale logo to style the top-most area of your sale page.

To define a new grouping, click the button "Add Another Group". This will create a group that contains the first section of your sale page. Once this exists, you'll see a button to "Configure Background".

Background Configuration Options

Each grouping can contain one or more section of content, and can have a background configured in a few different ways.
  • Background Gradient - you can select one solid color, or two colors to generate a gradient background color, and you can choose which direction the gradient should go
  • Background Image - You can upload one big image, or a smaller image with the intention of repeating horizontally, vertically, or both.

Adding Sections to a Grouping

Once you have defined a background grouping, you'll see a button to move the next content section into that grouping. Use that button to move any subsequent sections into your grouping that you wish to be presented in front of the same background.

Removing a Background Grouping

You can only remove the last background grouping on the page. For the last grouping, you will see a button that says "Remove this group".

Background Grouping With Tabs

Groupings will always end above a 'tab' section if present on your page. Any sections below the tabs, including the tabs themselves, will require a background grouping to be defined for each selected tab.
