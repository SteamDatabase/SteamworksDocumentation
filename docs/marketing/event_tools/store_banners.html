Sale Page Editor - Store Page Banners

Store Page Banners are special event banners that can appear across the top of a game's store page, just below the product name. Here is some information on how those banners work and how to set them up.

Types of banners

There are three ways that banners can appear on the top of a game store page, and they function differently. Here are some details.

  1. Game Update Banner: A banner that comes from an individual update event and opens the event details over the game's store page
  2. Sale Event Banner: A banner that comes from participation of that game on a sale page. Clicking this banner would go to the sale page.
  3. Halloween Banner: There is a special Halloween banner that can appear on the top of participating games' store pages during the Steam Halloween sale

Game Update Banner

A banner that comes from an individual update event and opens the event details over the game's store page. This is useful when a single game has big news or a big update that we want to amplify on that game's store page.

User Experience:

Clicking on this banner would open an event details panel that floats over the rest of the game page content. The player can scroll the panel, read the news, interact with links, etc. Or simply close the news and be back on the store page for the game.

Uses:

  • If we're promoting a big update to a game, say via a takeover or IM, then we want the update news to be obvious and easy to open on the store page
  • Even if we're not promoting it, a game may have a huge update that is their biggest news of the year and they want to make a big deal of it

Setting up:

  1. The partner must first create an event of any type other than "Cross Promotion"
  2. Then, the partner must upload a 'Library Spotlight' image to that event. By default, this image will be set to appear in the library page for the game just below the 'play' button. But it can be disabled by toggling the switch right above the banner image in the event editor.
  3. Once the banner image is provided, Valve can enable showing the banner by toggling an option under the 'options' tab.

  4. The banner will then appear on the top of the game page starting with the 'start visibility' date of the event, and ending after two weeks.

Sale Event Banner

A banner that comes from participation of that game on a sale page. Clicking this banner would go to the sale page. This is useful when running an event with multiple games that you want to link back to a sale landing page.

User Experience:

The banner will appear on all of the games included on the sale page. When a player clicks on the banner, it will take them to the sale page

Uses:

  • If running a publisher weekend and want all the included games to link back to the main event
  • If running a franchise sale and want all the included games to link back to the main event
  • 3rd party event organizers often set up a banner so the included games will link back to the main event page

Setting up:

  1. Valve must first enable the sale page feature under a Steam Community Group (usually be their publisher/developer/franchise page) : Create New Sale Page Wizard
  2. Once a sale page is set up, "Store page banner" assets can be added to that sale page (requires both desktop and mobile versions)
  3. If those assets are uploaded, then we'll show the banner on store pages for all included games
  4. The banner will start with the start visibility date set for the sale page, and end two weeks later, or with the end of the associated sale page.

Halloween Event Banner

During the Steam Halloween sales event, you may see a special Halloween banner that appears on the top of the store page for each participating game. This special banner is enabled by Valve and only lasts during the Steam Halloween Sales event. Clicking the banner will take players back to the Halloween sale page.

Frequently Asked Questions

Q. What if there are multiple banners that could show up on a game's store page? Which one will appear?
A. There are three possible banners that can appear at the top of a store page. Here is the order of priority.
  1. During the Steam Halloween sale, we sometimes show a special banner on participating games. This will always take priority during the Steam Halloween sale.
  2. A Game Update Banner will take next priority
  3. An event organized under the linked developer or publisher homepage will take next priority
  4. If there are multiple Sale Event Banners, then Steam will show the one with the latest valid start date

Q. What if my game is listed on a sale page and I don't want the banner on my store page?
A. We don't have controls yet for hiding the banner from specific game store pages. If your game is part of a sales event that has a store page banner enabled, it will appear on all the participating games' store pages.
