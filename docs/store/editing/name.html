Changing Your Game's Name

You can change the name of your game within Steamworks at any time until your store page has been through the pre-release review process.

Where your game's name is defined

Your game's name is defined in three main locations:
  • Store info - On the "Basic Info" tab under "Edit Store". This name is used within the Steam store, including search results.
  • Steamworks Info - On the "General Application Settings" tab under Steamworks Settings. This name is used within the Steam library and the Steam Community.
  • Package Info - Each store package that sells your game. This is used to present your purchase option shown on your store page and during the checkout process.

Localizing your game's name

You can provide localizations for your game name. Please see: Localized App name

Changing the name of a released game

Once your store page has passed through the store page review process, you won't be able to change the name of your game unless you have unlocked Profile Features

If you need to change the name of your game or add localized versions, please create a support ticket and let us know the game you are changing along with the names you wish to have it set to.
