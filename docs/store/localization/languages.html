Languages Supported on Steam

For more details on managing localization of your game and store presence on Steam, please see Localization and Languages

There are two primary levels of support for languages on Steam:
  1. Full Platform Support - This set of languages is available for players to select within Steam to run the Steam client in and browse the store in. We also translate the Steam client UI, Store, and Community into these languages so players have a fully native language experience in Steam. This is the only set of languages supported by Steamworks APIs.
  2. Game Support Only - This set of languages is available for developers to indicate the languages their game supports. This indication will appear on the game's store page and be searchable by players after October, 2022. Please see New Language Support announcement for details.

Frequently Asked Questions

  • Q: What if my game doesn't include any of the "Full Platform Supported" languages?

    A: If your game doesn't include any of the "Full Platform Supported" languages, you'll need to make sure you enter your store written descriptions, art capsules, etc. under "English" in the store editor drop-downs. English is Steam's fallback language and needs to have content, even if that content is actually another language.
