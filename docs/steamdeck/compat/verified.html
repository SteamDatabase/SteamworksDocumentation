Steam Deck Verified Landing Pages

Steam automatically creates a special landing page for any game that is marked as Steam Deck Verified.

This page displays the first trailer available on the game's store page, based on any language or country restrictions set for the trailers.

See your game's page

By adding the appID to the end of a URL, you can easily view and share this page for any of your games that are Steam Deck Verified.

The URL format for this page is as follows:
https://store.steampowered.com/verified/<appid>
